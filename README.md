# LoadTester

Official public repository for **LoadTester**.

LoadTester is a modern platform for **HTTP load testing** and **application load testing**. It helps teams run tests quickly, monitor latency, throughput, and failures in real time, and integrate performance checks into CI/CD workflows.

## Website
- Main site: https://loadtester.org/
- App: https://app.loadtester.org/

## What LoadTester does
- Run load tests without managing test infrastructure
- Test APIs, websites, login flows, checkout flows, and webhooks
- Monitor live analytics during test execution
- Configure thresholds and auto-stop conditions
- Trigger tests from automation and CI/CD pipelines

## Who it’s for
LoadTester is built for developers, QA teams, DevOps engineers, and product teams who want fast, repeatable performance validation before release.
